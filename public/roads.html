Which roads around Croke Park close to through-traffic on a big match day, and which routes keep resident-pass access — this is the same core set of streets every event, so it lives on one page instead of repeating under every match.

Roads/access routes come from Croke Park's own community notice for the nearest upcoming event. Cordons sit at specific junctions along a route, not evenly along its whole length — this map shows which streets are affected, not exactly where the barriers go.
